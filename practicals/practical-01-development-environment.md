## Теоретическая часть

Python — это интерпретируемый язык программирования, что означает, что программы на Python выполняются специальной программой-интерпретатором. Для написания и запуска программ на Python необходима установленная среда разработки.

### Основные элементы среды разработки:
1. Интерпретатор Python
2. Редактор кода
3. Средства отладки (опционально)

### Функции ввода и вывода:
- `print()` — выводит данные на экран
- `input()` — получает данные от пользователя

Пример простой программы:
```python
print("Привет, мир!")
name = input("Как вас зовут? ")
print("Рад знакомству,", name)
```

### Задание 3: Приветствие пользователя
Напишите программу, которая запрашивает имя пользователя и выводит приветствие.

**Пример выполнения:**
```
Как вас зовут? Алексей
Привет, Алексей!
```

**Шаблон кода:**
```python
# Запросить имя пользователя
name = input("Как вас зовут? ")

# Вывести приветствие
print("Привет,", name + "!")
```

### Задание 4: Калькулятор приветствий
Создайте программу, которая запрашивает имя и возраст пользователя, а затем выводит приветствие с указанием возраста.

**Пример выполнения:**
```
Как вас зовут? Мария
Сколько вам лет? 20
Привет, Мария! Вам 20 лет.
```

**Шаблон кода:**
```python
# Запросить имя и возраст
name = input("Как вас зовут? ")
age = input("Сколько вам лет? ")

# Вывести приветствие с возрастом
print("Привет,", name + "! Вам", age, "лет.")
```

### Задание 5: Простейшие вычисления
Создайте программу, которая запрашивает у пользователя два числа и выводит их сумму.

**Пример выполнения:**
```
Введите первое число: 5
Введите второе число: 3
Сумма чисел: 8
```

**Шаблон кода:**
```python
# Запросить два числа
num1 = int(input("Введите первое число: "))
num2 = int(input("Введите второе число: "))

# Вычислить сумму
sum = num1 + num2

# Вывести результат
print("Сумма чисел:", sum)
```

### Задание 6: Конвертер температуры
Создайте программу, которая конвертирует температуру из Цельсия в Фаренгейт по формуле: F = C × 9/5 + 32

**Пример выполнения:**
```
Введите температуру в градусах Цельсия: 25
Температура в Фаренгейтах: 77.0
```

**Шаблон кода:**
```python
# Запросить температуру в Цельсиях
celsius = float(input("Введите температуру в градусах Цельсия: "))

# Конвертировать в Фаренгейты
fahrenheit = celsius * 9/5 + 32

# Вывести результат
print("Температура в Фаренгейтах:", fahrenheit)
```

### Задание 7: Генератор имен
Создайте программу, которая просит пользователя ввести прилагательное и существительное, а затем генерирует забавное имя для питомца.

**Пример выполнения:**
```
Введите прилагательное: веселый
Введите существительное: кот
Ваш питомец может зваться: веселый кот
```

**Шаблон кода:**
```python
# Запросить прилагательное и существительное
adjective = input("Введите прилагательное: ")
noun = input("Введите существительное: ")

# Сгенерировать имя для питомца
pet_name = adjective + " " + noun

# Вывести результат
print("Ваш питомец может зваться:", pet_name)
```

### Задание 8: Программа-анкета
Создайте программу, которая задает несколько вопросов пользователю и затем выводит сводку по ответам.

**Пример выполнения:**
```
Добро пожаловать! Пожалуйста, ответьте на несколько вопросов.
Как вас зовут? Елена
Сколько вам лет? 22
Где вы живете? Москва
---
Имя: Елена
Возраст: 22
Город: Москва
```

**Шаблон кода:**
```python
print("Добро пожаловать! Пожалуйста, ответьте на несколько вопросов.")

# Запросить информацию у пользователя
name = input("Как вас зовут? ")
age = input("Сколько вам лет? ")
city = input("Где вы живете? ")

# Вывести сводку
print("---")
print("Имя:", name)
print("Возраст:", age)
print("Город:", city)
```
