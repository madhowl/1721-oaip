## Цель работы

Научиться работать с основными типами данных в Python (числовые типы, строки, логический тип), создавать переменные, выполнять операции с данными и преобразовывать типы.

## Теоретическая часть

Python - язык с динамической типизацией, где тип переменной определяется автоматически при присваивании значения. Основные типы данных:
- `int` - целые числа
- `float` - числа с плавающей точкой
- `str` - строки
- `bool` - логический тип (True/False)

Создайте переменные следующих типов:

### Задание 2: Операции с числовыми типами
Создайте две переменные:
- `a = 17`
- `b = 5`

Выполните и выведите результаты следующих операций:
- Сложение: `a + b`
- Вычитание: `a - b`
- Умножение: `a * b`
- Деление: `a / b`
- Целочисленное деление: `a // b`
- Остаток от деления: `a % b`
- Возведение в степень: `a ** b`

**Код для выполнения:**
```python
# Создание переменных

b = 5

# Выполнение операций
print(f"a + b = {a + b}")
print(f"a - b = {a - b}")
print(f"a * b = {a * b}")
print(f"a / b = {a / b}")
print(f"a // b = {a // b}")
print(f"a % b = {a % b}")
print(f"a ** b = {a ** b}")
```

### Задание 3: Работа со строками
Создайте строковые переменные:
- `first_name` с вашим именем
- `last_name` с вашей фамилией

Выполните следующие операции:
- Объедините имя и фамилию в одну строку через пробел
- Создайте строку, повторив ваше имя 3 раза
- Выведите первый и последний символ вашего полного имени
- Выведите длину полного имени

**Код для выполнения:**
```python
# Создание строковых переменных
first_name = "Иван"
last_name = "Петров"

# Объединение имени и фамилии
full_name = first_name + " " + last_name
print(f"Полное имя: {full_name}")

# Повторение имени
repeated_name = first_name * 3
print(f"Повторенное имя: {repeated_name}")

# Первый и последний символ полного имени
first_char = full_name[0]
last_char = full_name[-1]
print(f"Первый символ: {first_char}, последний символ: {last_char}")

# Длина полного имени
name_length = len(full_name)
print(f"Длина полного имени: {name_length}")
```

### Задание 4: Методы строк
Создайте строку `text = "  Привет, мир!  "` и примените к ней следующие методы:
- `upper()` - для преобразования в верхний регистр
- `lower()` - для преобразования в нижний регистр
- `strip()` - для удаления пробелов в начале и конце строки
- `replace("и", "е")` - для замены букв "и" на "е"
- `split()` - для разделения строки на слова

Выведите результаты всех операций.

**Код для выполнения:**
```python
# Создание строки
text = "  Привет, мир!  "

# Применение методов строк
print(f"Исходная строка: '{text}'")
print(f"В верхнем регистре: '{text.upper()}'")
print(f"В нижнем регистре: '{text.lower()}'")
print(f"Без пробелов по краям: '{text.strip()}'")
print(f"Замена 'и' на 'е': '{text.replace('и', 'е')}'")
print(f"Разделение на слова: {text.strip().split()}")
```

### Задание 5: Логические операции
Создайте переменные:
- `age = 20`
- `has_license = True`
- `is_student = True`

Вычислите и выведите результаты следующих логических выражений:
- Может ли человек водить машину: `age >= 18 and has_license`
- Является ли человек подростком или взрослым: `(13 <= age <= 19) or (age >= 20)`
- Является ли человек не студентом: `not is_student`
- Совершеннолетний студент: `age >= 18 and is_student`

**Код для выполнения:**
```python
# Создание переменных
age = 20
has_license = True
is_student = True

# Вычисление логических выражений
can_drive = age >= 18 and has_license
is_teen_or_adult = (13 <= age <= 19) or (age >= 20)
is_not_student = not is_student
adult_student = age >= 18 and is_student

# Вывод результатов
print(f"Может водить машину: {can_drive}")
print(f"Подросток или взрослый: {is_teen_or_adult}")
print(f"Не студент: {is_not_student}")
print(f"Совершеннолетний студент: {adult_student}")
```

### Задание 7: Неявное преобразование типов
Объясните, почему в следующем выражении результат равен 8.2:
```python
result = 5 + 3.2
print(result)
print(type(result))
```

**Ответ:**
В этом выражении происходит неявное преобразование типов. Целое число 5 автоматически преобразуется в число с плавающей точкой 5.0, чтобы выполнить операцию сложения с 3.2. Результатом становится число с плавающей точкой 8.2. Тип результата - float.

**Код для выполнения:**
```python
result = 5 + 3.2
print(result)
print(type(result))
```

### Задание 11: Безопасное преобразование
Реализуйте функцию `safe_int_convert(value)`, которая принимает строку и возвращает целое число, если строка может быть преобразована, и `None`, если преобразование невозможно. Протестируйте функцию на строках "123" и "abc".

**Код для выполнения:**
```python
def safe_int_convert(value):
    try:
        return int(value)
    except ValueError:
        return None

# Тестирование функции
print(safe_int_convert("123"))  # 123
print(safe_int_convert("abc"))  # None
```

### Задание 12: Арифметические выражения
Вычислите значение выражения: `(2 + 3.5) * 2 - 8 / 4` и объясните, какие преобразования типов происходят в этом выражении.

**Ответ:** 
Результат выражения: `(2 + 3.5) * 2 - 8 / 4 = 5.5 * 2 - 2.0 = 11.0 - 2.0 = 9.0`
В выражении происходит неявное преобразование целого числа 2 в 2.0 (float) при сложении с 3.5, затем результат (5.5) снова преобразуется к float при умножении, и 8 делится на 4 как float (2.0).

**Код для выполнения:**
```python
result = (2 + 3.5) * 2 - 8 / 4
print(f"Результат: {result}")
print(f"Тип результата: {type(result)}")
```
