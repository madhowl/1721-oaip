# Практическая работа 3: Условные операторы

## Цель работы
Освоить использование условных операторов в Python для решения задач с ветвлением:
- Определение четности чисел
- Сравнение чисел
- Создание калькулятора с выбором операции

## Теоретическая часть
Условные операторы позволяют программе выполнять разные действия в зависимости от различных условий. В Python условные конструкции реализуются с помощью операторов `if`, `elif` и `else`, которые позволяют организовать ветвление логики программы.

### Базовый синтаксис:
```python
if условие:
    # код, который выполняется если условие истинно
elif другое_условие:
    # код, который выполняется если другое_условие истинно
else:
    # код, который выполняется если все условия ложны
```

### Задание 1: Определение четности числа
Напишите программу, которая запрашивает у пользователя целое число и определяет, является ли оно четным или нечетным.

**Пример выполнения:**
```
Введите число: 12
Число 12 является четным
```
или
```
Введите число: 7
Число 7 является нечетным
```

**Подсказка:** Для проверки четности используйте оператор `%` (остаток от деления). Если `число % 2 == 0`, то число четное.

**Код для выполнения:**
```python
# Запрос числа у пользователя
number = int(input("Введите число: "))

# Проверка четности
if number % 2 == 0:
    print(f"Число {number} является четным")
else:
    print(f"Число {number} является нечетным")
```

### Задание 2: Сравнение двух чисел
Напишите программу, которая запрашивает у пользователя два числа и выводит, какое из них больше, меньше или они равны.

**Пример выполнения:**
```
Введите первое число: 5
Введите второе число: 8
8 больше чем 5
```
или
```
Введите первое число: 3
Введите второе число: 3
Числа равны
```

**Код для выполнения:**
```python
# Запрос двух чисел
num1 = int(input("Введите первое число: "))
num2 = int(input("Введите второе число: "))

# Сравнение чисел
if num1 > num2:
    print(f"{num1} больше чем {num2}")
elif num1 < num2:
    print(f"{num2} больше чем {num1}")
else:
    print("Числа равны")
```

### Задание 3: Калькулятор с выбором операции
Создайте простой калькулятор, который:
1. Запрашивает два числа
2. Запрашивает операцию (+, -, *, /)
3. Выполняет выбранную операцию и выводит результат
4. Обрабатывает ошибку деления на ноль

**Пример выполнения:**
```
Введите первое число: 10
Введите второе число: 5
Выберите операцию (+, -, *, /): *
Результат: 10 * 5 = 50
```

**Дополнительно:**
- Обработайте случай деления на ноль
- Обработайте неправильный ввод операции

**Код для выполнения:**
```python
# Запрос чисел
num1 = float(input("Введите первое число: "))
num2 = float(input("Введите второе число: "))

# Запрос операции
operation = input("Выберите операцию (+, -, *, /): ")

# Выполнение операции
if operation == '+':
    result = num1 + num2
    print(f"Результат: {num1} + {num2} = {result}")
elif operation == '-':
    result = num1 - num2
    print(f"Результат: {num1} - {num2} = {result}")
elif operation == '*':
    result = num1 * num2
    print(f"Результат: {num1} * {num2} = {result}")
elif operation == '/':
    if num2 != 0:
        result = num1 / num2
        print(f"Результат: {num1} / {num2} = {result}")
    else:
        print("Ошибка: деление на ноль!")
else:
    print("Неподдерживаемая операция")
```

### Задание 4: Оценка успеваемости

Напишите программу, которая определяет, является ли введенный год високосным.
Год является високосным, если:
- Он делится на 4 И (не делится на 100 ИЛИ делится на 40)

Введите год: 2024

### Задание 6: Система скидок (повышенная сложность)
Представьте, что вы создаете систему скидок для магазина:
- Если сумма покупки >= 1000 и покупатель участник клуба - скидка 15%
- Если сумма покупки >= 1000 и покупатель не участник клуба - скидка 10%
- Если сумма покупки >= 500 и покупатель участник клуба - скидка 8%
- Если сумма покупки >= 500 и покупатель не участник клуба - скидка 5%
- Если сумма покупки < 500 - скидка 0%

**Пример выполнения:**
```
Введите сумму покупки: 800
Вы являетесь участником клуба? (да/нет): да
Ваша скидка: 8.0%
Итоговая сумма: 736.00
```

**Код для выполнения:**
```python
# Запрос данных
purchase_amount = float(input("Введите сумму покупки: "))
club_member_input = input("Вы являетесь участником клуба? (да/нет): ")
is_club_member = club_member_input.lower() == 'да'

# Определение скидки
if purchase_amount >= 1000:
    if is_club_member:
        discount = 0.15  # 15% скидка для участников
    else:
        discount = 0.10  # 10% скидка для обычных клиентов
elif purchase_amount >= 500:
    if is_club_member:
        discount = 0.08  # 8% скидка для участников
    else:
        discount = 0.05  # 5% скидка для обычных клиентов
else:
    discount = 0  # Нет скидки

# Вычисление итоговой суммы
final_amount = purchase_amount * (1 - discount)

# Вывод результатов
print(f"Ваша скидка: {discount*100}%")
print(f"Итоговая сумма: {final_amount:.2f}")
```

### Задание 7: Определение типа треугольника
Напишите программу, которая запрашивает у пользователя длины трех сторон треугольника и определяет тип треугольника:
- Равносторонний (все стороны равны)
- Равнобедренный (две стороны равны)
- Разносторонний (все стороны разные)
- Проверьте, может ли существовать треугольник с такими сторонами

**Пример выполнения:**
```
Введите длину первой стороны: 5
Введите длину второй стороны: 5
Введите длину третьей стороны: 5
Треугольник равносторонний
```

**Код для выполнения:**
```python
# Запрос длин сторон
side1 = float(input("Введите длину первой стороны: "))
side2 = float(input("Введите длину второй стороны: "))
side3 = float(input("Введите длину третьей стороны: "))

# Проверка существования треугольника
if side1 + side2 > side3 and side1 + side3 > side2 and side2 + side3 > side1:
    # Определение типа треугольника
    if side1 == side2 == side3:
        print("Треугольник равносторонний")
    elif side1 == side2 or side1 == side3 or side2 == side3:
        print("Треугольник равнобедренный")
    else:
        print("Треугольник разносторонний")
else:
    print("Треугольник с такими сторонами не существует")
```

### Задание 8: Калькулятор BMI (Body Mass Index)
Напишите программу, которая вычисляет индекс массы тела и определяет состояние здоровья:
BMI = вес (кг) / (рост (м))²
- BMI < 18.5: Недостаточный вес
- 18.5 ≤ BMI < 25: Нормальный вес
- 25 ≤ BMI < 30: Избыточный вес
- BMI ≥ 30: Ожирение

**Пример выполнения:**
```
Введите ваш вес в кг: 70
Введите ваш рост в метрах: 1.75
Ваш BMI: 2.86
Состояние: Нормальный вес
```

**Код для выполнения:**
```python
# Запрос данных
weight = float(input("Введите ваш вес в кг: "))
height = float(input("Введите ваш рост в метрах: "))

# Вычисление BMI
bmi = weight / (height ** 2)

# Определение состояния
if bmi < 18.5:
    condition = "Недостаточный вес"
elif 18.5 <= bmi < 25:
    condition = "Нормальный вес"
elif 25 <= bmi < 30:
    condition = "Избыточный вес"
else:
    condition = "Ожирение"

# Вывод результатов
print(f"Ваш BMI: {bmi:.2f}")
print(f"Состояние: {condition}")
```
