## Цель работы

Научиться использовать списки в Python, освоить основные операции с ними: создание, добавление, удаление, изменение элементов, сортировка, поиск, а также применение списков для решения задач обработки данных.

## Теоретическая часть

Список - это упорядоченная изменяемая коллекция элементов в Python. Списки могут содержать элементы разных типов и поддерживают множество методов для работы с данными:

```python
# Создание списков
empty_list = []
numbers = [1, 2, 3, 4, 5]
mixed_list = [1, "hello", 3.14, True]

# Основные операции
numbers.append(6)        # Добавление элемента в конец
numbers.insert(0, 0)     # Вставка элемента по индексу
numbers.remove(3)        # Удаление первого вхождения элемента
popped = numbers.pop()   # Удаление и возврат последнего элемента
numbers.sort()           # Сортировка списка
numbers.reverse()        # Разворот списка
```

### Задание 1: Создание и вывод списка

Создайте программу, которая запрашивает у пользователя 5 чисел и сохраняет их в список. Затем выведите список на экран, его длину и сумму всех элементов.

**Пример выполнения:**
```
Введите число 1: 10
Введите число 2: 20
Введите число 3: 30

Введите число 5: 50
Список: [10, 20, 30, 40, 50]
Длина списка: 5
Сумма элементов: 150
```

**Код для выполнения:**
```python
numbers = []

    num = int(input(f"Введите число {i+1}: "))

print(f"Длина списка: {len(numbers)}")

```

### Задание 2: Поиск элемента в списке

Создайте программу, которая позволяет пользователю ввести список чисел, а затем искать конкретное число в этом списке. Программа должна выводить индекс первого вхождения числа или сообщение, что число не найдено.

**Пример выполнения:**
```
Введите числа через пробел: 5 10 15 20 25
Введите число для поиска: 15
Число 15 найдено по индексу 2
```

**Код для выполнения:**
```python
numbers_str = input("Введите числа через пробел: ")
numbers = [int(x) for x in numbers_str.split()]

target = int(input("Введите число для поиска: "))

try:
    index = numbers.index(target)
    print(f"Число {target} найдено по индексу {index}")
except ValueError:
    print(f"Число {target} не найдено в списке")
```

### Задание 4: Сортировка и фильтрация

Создайте программу, которая:
1. Запрашивает у пользователя список чисел
2. Сортирует его по возрастанию
3. Фильтрует числа больше 50
4. Выводит отсортированный и отфильтрованный список

**Пример выполнения:**
```
Введите числа через пробел: 75 30 90 15 60 45 80
```

### Задание 5: Удаление дубликатов

Создайте программу, которая удаляет дубликаты из списка, сохраняя порядок элементов.

**Пример выполнения:**
```
Введите числа через пробел: 1 2 3 2 4 1 5 3
Список без дубликатов: [1, 2, 3, 4, 5]
```

**Код для выполнения:**
```python
numbers_str = input("Введите числа через пробел: ")
numbers = [int(x) for x in numbers_str.split()]

# Удаление дубликатов с сохранением порядка
unique_numbers = []
seen = set()

for num in numbers:
    if num not in seen:
        unique_numbers.append(num)
        seen.add(num)

print(f"Список без дубликатов: {unique_numbers}")
```

### Задание 6: Обработка строковых списков

Создайте программу, которая:
1. Запрашивает у пользователя список слов
2. Находит самое длинное и самое короткое слово
3. Выводит слова в алфавитном порядке
4. Выводит слова, начинающиеся на гласную

**Пример выполнения:**
```
Введите слова через пробел: яблоко банан апельсин груша ананас
Самое длинное слово: апельсин
Самое короткое слово: груша
Слова в алфавитном порядке: ['ананас', 'апельсин', 'банан', 'груша', 'яблоко']
Слова, начинающиеся на гласную: ['ананас', 'апельсин']
```

**Код для выполнения:**
```python
words_str = input("Введите слова через пробел: ")
words = words_str.split()

# Находим самое длинное и самое короткое слово
if words:
    longest = max(words, key=len)
    shortest = min(words, key=len)
    
    print(f"Самое длинное слово: {longest}")
    print(f"Самое короткое слово: {shortest}")
    
    # Сортировка по алфавиту
    alphabetically_sorted = sorted(words)
    print(f"Слова в алфавитном порядке: {alphabetically_sorted}")
    
    # Слова, начинающиеся на гласную
    vowels = "аеёиоуыэюяaeiouAEIOU"
    vowel_words = [word for word in words if word[0] in vowels]
    print(f"Слова, начинающиеся на гласную: {vowel_words}")
```

### Задание 7: Статистика списка

Создайте программу, которая анализирует список чисел и выводит:
- Минимальное и максимальное значение
- Среднее арифметическое
- Медиану
- Моду (наиболее часто встречающееся значение)

**Пример выполнения:**
```
Введите числа через пробел: 1 2 3 2 4 2 5
Минимальное значение: 1
Максимальное значение: 5
Среднее арифметическое: 2.71
Медиана: 2
Мода: 2
```

**Код для выполнения:**
```python
from collections import Counter
import statistics

numbers_str = input("Введите числа через пробел: ")
numbers = [int(x) for x in numbers_str.split()]

if numbers:
    print(f"Минимальное значение: {min(numbers)}")
    print(f"Максимальное значение: {max(numbers)}")
    print(f"Среднее арифметическое: {sum(numbers)/len(numbers):.2f}")
    
    # Медиана
    sorted_nums = sorted(numbers)
    n = len(sorted_nums)
    if n % 2 == 0:
        median = (sorted_nums[n//2 - 1] + sorted_nums[n//2]) / 2
    else:
        median = sorted_nums[n//2]
    print(f"Медиана: {median}")
    
    # Мода
    counter = Counter(numbers)
    max_count = max(counter.values())
    modes = [num for num, count in counter.items() if count == max_count]
    print(f"Мода: {modes[0] if len(modes) == 1 else modes}")
```

### Задание 8: Работа с матрицами

Создайте программу, которая создает двумерный список (матрицу) 3x3, заполняет его случайными числами и выводит:
- Саму матрицу
- Сумму всех элементов
- Сумму главной диагонали
- Сумму побочной диагонали

**Пример выполнения:**
```
Матрица:
[[1, 2, 3],
 [4, 5, 6],
 [7, 8, 9]]
Сумма всех элементов: 45
Сумма главной диагонали: 15
Сумма побочной диагонали: 15
```

**Код для выполнения:**
```python
import random

# Создаем матрицу 3x3 со случайными числами
matrix = [[random.randint(1, 10) for _ in range(3)] for _ in range(3)]

print("Матрица:")
for row in matrix:
    print(row)

# Сумма всех элементов
total_sum = sum(sum(row) for row in matrix)
print(f"Сумма всех элементов: {total_sum}")

# Сумма главной диагонали
main_diagonal_sum = sum(matrix[i][i] for i in range(3))
print(f"Сумма главной диагонали: {main_diagonal_sum}")

# Сумма побочной диагонали
secondary_diagonal_sum = sum(matrix[i][2-i] for i in range(3))
print(f"Сумма побочной диагонали: {secondary_diagonal_sum}")
```
