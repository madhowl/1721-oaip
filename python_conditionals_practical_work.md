# Практическая работа: Условные операторы в Python

## Цель работы
Освоить использование условных операторов в Python для решения задач с ветвлением:
- Определение четности чисел
- Сравнение чисел
- Создание калькулятора с выбором операции

## Теоретическая часть
Условные операторы позволяют программе выполнять разные действия в зависимости от различных условий. В Python условные конструкции реализуются с помощью операторов `if`, `elif` и `else`, которые позволяют организовать ветвление логики программы.

### Базовый синтаксис:
```python
if условие:
    # код, который выполняется если условие истинно
elif другое_условие:
    # код, который выполняется если другое_условие истинно
else:
    # код, который выполняется если все условия ложны
```

## Задания

### Задание 1: Определение четности числа
Напишите программу, которая запрашивает у пользователя целое число и определяет, является ли оно четным или нечетным.

**Пример выполнения:**
```
Введите число: 12
Число 12 является четным
```
или
```
Введите число: 7
Число 7 является нечетным
```

**Подсказка:** Для проверки четности используйте оператор `%` (остаток от деления). Если `число % 2 == 0`, то число четное.

### Задание 3: Калькулятор с выбором операции
Создайте простой калькулятор, который:
1. Запрашивает два числа
2. Запрашивает операцию (+, -, *, /)
3. Выполняет выбранную операцию и выводит результат
4. Обрабатывает ошибку деления на ноль

**Пример выполнения:**
```
Введите первое число: 10
Введите второе число: 5
Выберите операцию (+, -, *, /): *
Результат: 10 * 5 = 50
```

**Дополнительно:**
- Обработайте случай деления на ноль
- Обработайте неправильный ввод операции
