# Лекция 2: Знакомство с языком Python

## Введение

Python — это высокоуровневый язык программирования общего назначения, который был создан Гвидо ван Россумом и впервые выпущен в 1991 году. Python стал одним из самых популярных языков программирования благодаря своей простоте, читаемости кода и широким возможностям. В этой лекции мы познакомимся с основами языка Python, его особенностями и средами разработки.

## Основное содержание

### Назначение языка Python

Python используется в самых разных областях программирования:

1. **Веб-разработка** — создание серверных частей веб-приложений с использованием фреймворков Django, Flask и других.
2. **Научные вычисления и анализ данных** — библиотеки NumPy, Pandas, Matplotlib и другие.
3. **Машинное обучение и искусственный интеллект** — TensorFlow, PyTorch, scikit-learn.
4. **Автоматизация задач** — написание скриптов для автоматизации рутинных операций.
5. **Разработка игр** — с использованием библиотеки pygame.
6. **Мобильные приложения** — с использованием Kivy и других фреймворков.
7. **Скрипты системного администрирования** — автоматизация задач на серверах.

Python особенно популярен среди новичков в программировании благодаря своей простоте и понятному синтаксису.

### Особенности языка Python

1. **Читаемость кода** — Python использует отступы для определения блоков кода, что делает структуру программы визуально понятной.
2. **Динамическая типизация** — типы переменных определяются во время выполнения, что упрощает написание кода.
3. **Интерпретируемый язык** — код выполняется интерпретатором, а не компилируется в машинный код.
4. **Объектно-ориентированное программирование** — поддержка классов и объектов.
5. **Функциональное программирование** — поддержка функций высшего порядка.
6. **Богатая стандартная библиотека** — Python поставляется с большим количеством встроенных модулей.
7. **Кроссплатформенность** — Python работает на различных операционных системах.

Для написания программ на Python можно использовать различные среды разработки:

2. **PyCharm** — мощная интегрированная среда разработки от JetBrains.
   - Автодополнение кода
   - Отладчик
   - Интеграция с Git
   - Подсветка синтаксиса
   - Среда разработки для больших проектов

3. **Visual Studio Code** — легковесный редактор кода с поддержкой Python через расширения.
   - Большое количество расширений
   - Интеграция с Git
   - Отладчик
   - Подсветка синтаксиса
   - Подходит для различных проектов

4. **Jupyter Notebook** — интерактивная среда, особенно популярная в научных вычислениях.
   - Возможность запускать код по частям
   - Вставка текста, формул и графиков
   - Подходит для исследовательского программирования

### Структура программы на Python

Программа на Python состоит из различных элементов:

1. **Комментарии** — строки, которые не выполняются интерпретатором. Комментарии начинаются с символа `#`.

```python
# Это комментарий
# Следующая строка выводит приветствие
print("Привет, мир!")
```

2. **Импорты модулей** — подключение внешних библиотек и модулей.

```python
import math
from datetime import datetime
```

3. **Определение переменных** — создание переменных и присвоение им значений.

```python
name = "Алексей"
age = 25
```

4. **Функции** — блоки кода, которые можно вызывать многократно.

```python
def greet(name):
    return f"Привет, {name}!"
```

5. **Классы** — шаблоны для создания объектов.

```python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

6. **Инструкции управления** — условные операторы, циклы и т.д.

```python
if age >= 18:
    print("Совершеннолетний")
else:
    print("Несовершеннолетний")
```

### Ввод и вывод данных

В Python для вывода данных используется функция `print()`:

```python
print("Это сообщение на экране")
print("Результат:", 42)
print(f"Переменная x = {x}")
```

Для ввода данных используется функция `input()`:

```python
name = input("Введите ваше имя: ")
age = int(input("Введите ваш возраст: "))
```

Обратите внимание, что `input()` всегда возвращает строку, поэтому при необходимости нужно преобразовать ввод в нужный тип данных.

### Основные соглашения о стиле кода (PEP 8)

PEP 8 — это руководство по стилю написания кода на Python. Вот основные рекомендации:

1. **Отступы** — используйте 4 пробела на каждый уровень отступа.
2. **Длина строки** — ограничьте длину строки 79 символами.
3. **Имена переменных** — используйте snake_case (например, `my_variable`).
4. **Имена классов** — используйте CamelCase (например, `MyClass`).
5. **Константы** — используйте ПРОПИСНЫЕ_БУКВЫ (например, `MAX_SIZE`).
6. **Пробелы вокруг операторов** — добавляйте пробелы вокруг бинарных операторов.
7. **Пустые строки** — используйте пустые строки для разделения функций и классов.

Пример правильно оформленного кода:

```python
# Правильно оформленный код
def calculate_area(length, width):
    """
    Вычисляет площадь прямоугольника
    :param length: длина
    :param width: ширина
    :return: площадь
    """
    area = length * width
    return area

# Использование функции
rectangle_length = 10
rectangle_width = 5
area = calculate_area(rectangle_length, rectangle_width)
print(f"Площадь прямоугольника: {area}")
```

## Практические примеры

Рассмотрим простую программу на Python, которая демонстрирует основные концепции:

```python
"""
Программа для вычисления факториала числа
"""

def factorial(n):
    """Вычисляет факториал числа n"""
    if n <= 1:
        return 1
    else:
        return n * factorial(n - 1)

def main():
    """Основная функция программы"""
    print("Программа для вычисления факториала")
    
    try:
        number = int(input("Введите положительное целое число: "))
        
        if number < 0:
            print("Факториал не определен для отрицательных чисел")
        else:
            result = factorial(number)
            print(f"Факториал числа {number} равен {result}")
            
    except ValueError:
        print("Ошибка: введите целое число")

# Запуск основной функции
if __name__ == "__main__":
    main()
```

Этот пример показывает:
- Использование комментариев и документации
- Определение функций
- Ввод данных от пользователя
- Обработку исключений
- Условные операторы
- Рекурсию
