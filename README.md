## Описание курса

Курс "Основы алгоритмизации и программирования" предназначен для студентов, начинающих изучение программирования. В рамках курса студенты осваивают базовые концепции алгоритмизации и программирования на примере языка Python. Курс включает в себя как теоретическую часть (лекции), так и практическую (лабораторные работы), что позволяет студентам не только понять основные принципы программирования, но и получить навыки написания реальных программ.

## Цели и задачи курса

### Цель курса:
Формирование у студентов базовых навыков алгоритмизации и программирования, а также развитие логического мышления, необходимого для решения задач в области информационных технологий.

### Задачи курса:
1. Ознакомить студентов с основами алгоритмизации и структурного программирования
2. Научить студентов использовать язык Python для решения различных задач
3. Развить навыки создания и анализа алгоритмов
4. Освоить основные структуры данных и алгоритмы
5. Научить студентов использовать современные инструменты разработки

## Требования к начальному уровню подготовки

Для успешного прохождения курса студент должен:
- Иметь базовые навыки работы с компьютером
- Уметь использовать текстовый редактор
- Понимать основы математики (школьный курс)

## Структура курса

Курс состоит из теоретической и практической частей. Общий объем курса составляет 64 академических часа (34 часа лекций и 30 часов практических занятий), что соответствует 2 зачетным единицам.

### Лекции

1. **[Лекция 1: Введение в алгоритмизацию](lectures/lecture-01-introduction-to-algorithms.md)** (2 часа)

   - Преобразование типов

5. **[Лекция 5: Циклы](lectures/lecture-05-loops.md)** (2 часа)

   - Регулярные выражения

7. **[Лекция 7: Списки и кортежи](lectures/lecture-07-lists-and-tuples.md)** (2 часа)

## Технические требования

Для прохождения курса необходимы:
- Компьютер с операционной системой Windows, macOS или Linux
- Установленный Python 3.8 или выше
- Текстовый редактор или IDE (рекомендуется PyCharm, VS Code или IDLE)
- Доступ к Интернету для установки дополнительных библиотек
