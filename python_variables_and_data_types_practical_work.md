## Краткая теоретическая справка

Python - язык с динамической типизацией, где тип переменной определяется автоматически при присваивании значения. Основные типы данных:
- `int` - целые числа
- `float` - числа с плавающей точкой
- `str` - строки
- `bool` - логический тип (True/False)
